Sobre o site:
- Contém três arquivos: index.html, functions.js e myStyle.css.
- Utilizei AngularJS, Jquery e Bootstrap.

Index.html
Desenvolvido em HTML5 e é composto de:
	- Menu Principal.
	- Secção de consultar atividades.
	- Secção de calcular impostos do regime tributário Simples Nacional.
	- Secção de calcular impostos do regime tributário Lucro Presumido.
	
Functions.js
	- Contém as chamadas para os webservices disponibilizados pela RESTFul API da empresa Contabilizei.
	- Tratativa de erro em caso de falha de algum webservice.

myStyle.css
	- Contém personalizações de layout para que o site fique com uma aparência agradável.
